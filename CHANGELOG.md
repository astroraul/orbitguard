## [0.1.0] - 2026-05-08

### Added
- Initial public release of OrbitGuard AI dashboard.
- 3D LEO globe powered by Three.js r128.
- Live TLE ingestion from CelesTrak.
- MMOD flux calculator (parametric model v1).
- Breakup events feed.
- Asset inspector with AI threat score (v3.1 heuristic).
- Mission report export.
- GitHub Pages deployment.
